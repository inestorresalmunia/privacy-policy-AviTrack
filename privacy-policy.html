🔒 Política de Privacidad - AviTrack

📅 Última actualización: Septiembre 2025 • Versión 1.2

📱 Información de la Aplicación

Nombre: AviTrack

Función: Escáner de códigos QR y códigos de barras con validación de ubicación GPS

Versión actual: 1.0.3

🎯 ¿Qué hace nuestra aplicación?

AviTrack es una herramienta profesional que permite:

  • ✅ Escanear códigos QR y códigos de barras
  • 📍 Validar ubicaciones mediante coordenadas GPS
  • 📊 Generar informes de verificación
  • 💾 Almacenar datos localmente hasta su envío al servidor
  • 🔄 Sincronizar datos automáticamente con sistemas corporativos

🛡️ Permisos Necesarios y Su Justificación

📷 Permiso de Cámara

¿Por qué lo necesitamos?

Para capturar imágenes de códigos QR y códigos de barras que contienen información de identificación y coordenadas GPS.

¿Cómo lo usamos?

  • Se activa únicamente cuando presionas el botón de escanear
  • Las fotos se procesan localmente en tu dispositivo
  • Solo guardamos las imágenes necesarias para el proceso de verificación
  • No accedemos a tu galería de fotos ni a otras imágenes

📍 Permisos de Ubicación (GPS)

¿Por qué lo necesitamos?

Para verificar que te encuentres en la ubicación correcta según las coordenadas embebidas en el código QR escaneado.

¿Cómo lo usamos?

  • Obtenemos tu ubicación solo durante el proceso de escaneo
  • Comparamos tu ubicación actual con las coordenadas del código
  • Calculamos la distancia para validar la precisión
  • No rastreamos tu ubicación continuamente
  • No compartimos datos de ubicación con terceros

💾 Almacenamiento

¿Por qué lo necesitamos?

Para guardar temporalmente las imágenes escaneadas y los datos de sesión hasta que se envíen al servidor corporativo.

¿Qué almacenamos?

  • Imágenes de códigos procesados
  • Coordenadas GPS y timestamps
  • Resultados de validación
  • Identificadores únicos del dispositivo

🌐 Acceso a Internet

¿Por qué lo necesitamos?

Para enviar los datos recopilados al servidor central y obtener direcciones a partir de coordenadas GPS.

¿Qué datos enviamos?

  • Códigos escaneados e información de validación
  • Coordenadas GPS (solo las relacionadas con el trabajo)
  • Timestamps y identificadores de dispositivo
  • No enviamos información personal o privada

🔒 Medidas de Seguridad

Implementamos las siguientes medidas para proteger tu información:

🔐 Cifrado de datos 🆔 ID único de dispositivo 🗑️ Eliminación automática 🚫 No compartimos con terceros 🏠 Procesamiento local
  • Identificación única del dispositivo para evitar duplicados
  • Validación de datos antes del envío al servidor
  • Eliminación automática de datos locales después del envío exitoso
  • Procesamiento local de todas las imágenes
  • Conexiones seguras para el envío de datos

❌ Lo que NO hacemos

  • ❌ No accedemos a tus contactos, mensajes o llamadas
  • ❌ No recopilamos información personal identificable
  • ❌ No compartimos datos con empresas de publicidad
  • ❌ No utilizamos datos para marketing o publicidad
  • ❌ No accedemos a otras aplicaciones instaladas
  • ❌ No grabamos audio ni video
  • ❌ No rastreamos tu ubicación fuera del horario de trabajo

👤 Tus Derechos como Usuario

  • 🚫 Denegar permisos: Puedes rechazar cualquier permiso (la app te informará sobre funcionalidades limitadas)
  • 👀 Revisar datos: Puedes ver todos los datos almacenados localmente desde la aplicación
  • 🗑️ Eliminar datos: Puedes borrar todos los datos desde la configuración de Android
  • 📱 Desinstalar: Puedes eliminar la aplicación completamente en cualquier momento
  • 📧 Contactar: Puedes contactarnos para resolver cualquier duda sobre privacidad

🏢 Uso Empresarial

Esta aplicación está diseñada para uso empresarial y profesional:

  • Los datos se envían a servidores corporativos seguros
  • Solo personal autorizado puede acceder a los datos
  • Los datos se utilizan únicamente para fines de verificación y control de activos
  • Cumplimos con todas las normativas de protección de datos aplicables
